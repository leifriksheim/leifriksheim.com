Git is hard: screwing up is easy, and figuring out how to fix your mistakes is fucking impossible.

## Install

```bash
npm install -g ohshitgit
```

## Usage

Once that's done, you can run this command inside your project's root directory:

```bash
ohshit
```

That's it! :tada:
